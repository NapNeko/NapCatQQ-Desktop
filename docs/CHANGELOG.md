## v2.0.0 (2025-04-02)

### ✨ 新增
- **API 调试功能**：全新接口文档调试工作台，支持请求构建、响应展示、Schema 可视化
- **一键发布脚本** (`release.py`)：本地一键完成版本更新、AI 生成日志、打 tag、推送
- **AI 更新日志生成工具**：基于 AI 自动生成结构化更新日志
- **MSI 安装包自动更新**：支持检测更新、下载、静默安装，完成应用内更新闭环

### 🔧 优化
- **重构发布流程**：统一收敛为 MSI 安装包发布，移除便携版构建逻辑
- **更新日志改进**：AI 驱动的日志生成，智能分类合并 commit
- **API 调试页面**：拆分模块化组件，优化布局和交互体验
- **Bot 配置系统**：配置模型增强，UI 组件修复和完善

### 🐛 修复
- **MSI 更新链路**：修复 Batch 脚本参数注入、提权命令、进程等待等关键问题
- **安装类型检测**：增强注册表权限处理，修复 Windows 路径大小写敏感问题
- **下载器**：增加原子写入 (.part 文件)，细化异常处理
- **PyInstaller**：修复入口文件路径错误
- **桌面更新页面**：优化进程退出等待逻辑，修复资源泄漏

### 📝 重要变更
- 安装方式统一改为 MSI 安装包
- 默认安装位置：`C:\Program Files\NapCatQQ Desktop\`
- 配置文件存储在：`%ProgramData%\NapCatQQ-Desktop\`
- 支持应用内自动更新

### 📝 使用须知

**⚠️ 系统要求与注意事项**  
- 🖥️ **兼容系统**：仅支持 Windows Server 2016 / Windows 10 及以上版本  
- 📦 **安装方式**：MSI 安装包（推荐）  
  - 下载 `NapCatQQ-Desktop-{版本}-x64.msi` 双击安装  
  - 默认安装位置：`C:\Program Files\NapCatQQ Desktop\`  
  - 支持应用内自动更新，无需手动下载安装包  
- ⚠️ **特别注意**：  
  - 首次运行时请仔细阅读 EULA 协议  
  - 建议以管理员身份运行以获得完整权限  
  - 配置文件存储在 `%ProgramData%\NapCatQQ-Desktop\`  

💡 **升级提示**：v2.0.0 起统一使用 MSI 安装包，旧版便携版用户请手动迁移配置后重新安装！

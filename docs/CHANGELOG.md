## 📝 使用须知

**⚠️ 系统要求与注意事项**  
- 🖥️ **兼容系统**：仅支持 Windows Server 2016 / Windows 10 及以上版本  
- 📦 **安装方式**：MSI 安装包（推荐）  
  - 下载 `NapCatQQ-Desktop-{版本}-x64.msi` 双击安装  
  - 默认安装位置：`C:\Program Files\NapCatQQ Desktop\`  
  - 支持应用内自动更新，无需手动下载安装包  
- ⚠️ **特别注意**：  
  - 首次运行时请仔细阅读 EULA 协议  
  - 建议以管理员身份运行以获得完整权限  
  - 配置文件存储在 `%ProgramData%\NapCatQQ-Desktop\`  

💡 **升级提示**：v2.0.0 起统一使用 MSI 安装包，旧版便携版用户请手动迁移配置后重新安装！

## 📝 使用须知

**⚠️ 系统要求与注意事项**  
- 🖥️ **兼容系统**：仅支持 Windows Server 2016 / Windows 10 及以上版本  
- 📂 **安装建议**：  
  - 请将软件放置于目标管理文件夹内 **再运行**（将自动生成配置文件于同级目录）  
  - ❗ 避免直接放在桌面或系统目录运行  
- ⚠️ **特别注意**：  
  - 首次运行时请关闭杀毒软件（如有误报）  
  - 建议右键「以管理员身份运行」获得完整权限  

💡 小提示：配置文件生成后，移动主程序时请同步移动配置文件！  

## 什么是累积更新
在当前大版本（如 v1.6）中，小版本号（如 v1.6.1）的更新是累积进行的。每个小版本均基于前一个小版本构建，即 v1.6.1 包含 v1.6.0 的全部更新内容，并在此基础之上添加本次的新增内容与修复。

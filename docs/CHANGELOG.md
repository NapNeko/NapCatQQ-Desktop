# 🚀 NapCatQQ Desktop 更新日志

## v2.0.3 (2025-04-03)

Tips: v2.0开始是破坏性更新，旧版无法直接更新，请手动下载更新喔，安装完成后到设置里迁移配置就可以了（直接把旧版文件夹往里拖就可以了）

### ✨ 主要新功能

#### 1. API 调试工作台
- 全新接口文档调试功能，支持请求构建、响应展示、Schema 可视化
- 拆分模块化组件：目录面板、详情面板、调试卡片、Schema 卡片
- 新增 HTTP 方法徽章组件、搜索对话框、公共工具模块
- 统一文案术语从"接口调试"改为"接口文档"
- 支持工作台状态持久化、重试机制和错误处理

#### 2. MSI 安装与自动更新
- 统一使用 MSI 安装包发布，移除便携版
- 应用内自动更新：检测、下载、静默安装完整闭环
- 安装类型识别（MSI vs 便携版）和迁移支持
- 更新脚本增强：参数注入、提权命令、进程等待、错误处理
- 临时目录回退机制，解决权限不足问题

### 🔧 系统重构与优化

#### 项目结构重构
- 重构 `bot_page`、`setup_page`、`home_page`、`component_page` 模块结构
- 清理冗余导出定义，移除多个模块的 `__all__` 声明
- 删除废弃的更新脚本、文档目录和过时文件
- 更新核心模块和 UI 层导入路径

#### 配置系统增强
- 配置模型增强，支持事务写入
- Bot 配置系统完善，支持配置保存前校验
- 新增遗留配置导入兼容模块，支持从旧版本迁移
- 新增拖拽文件夹组件、遗留导入对话框

#### 日志系统升级
- 新增 TRACE 日志级别，支持详细诊断输出
- 代码编辑器增强：日志高亮、平滑滚动、主题适配
- 新增崩溃通知中心模块，支持脱敏崩溃诊断包导出
- 日志模块集成通知中心，支持实时输出通知

#### UI/UX 改进
- 主题系统：支持 QFluentWidgets 主题配置绑定
- 新增骨架屏组件，优化加载体验
- 通知服务增强，支持通知时间线展示
- 桌面日志页面，支持日志实时查看和高亮

### 🐛 关键修复

#### 构建与打包
- 修复 PyInstaller 入口路径错误
- 修复 Windows PowerShell 命令解析问题
- 优化 PyInstaller 构建：集合过滤器、资源准备、图像处理钩子

- 修复下载器完成信号与状态校验问题

#### 安装类型检测

- 修复 Windows 路径大小写敏感问题
- 使用 `os.path.normcase` 替代 `resolve()` 避免符号链接副作用

#### UI 修复
- 修复跨平台字体渲染和高 DPI 显示问题
- 修复删除 Bot 时的状态清理
- 修复配置加载与事务写入问题
- 修复空 JSON 配置回填问题

### 📝 重要变更与迁移指南

#### 安装方式变更
| 项目 | 旧版（v1.7.x） | 新版（v2.0.0+） |
|------|---------------|----------------|
| 安装包 | ZIP 便携版 | MSI 安装包 |
| 安装位置 | 任意文件夹 | `C:\Program Files\NapCatQQ Desktop\` |
| 配置存储 | 应用目录 | `%ProgramData%\NapCatQQ-Desktop\` |
| 更新方式 | 手动下载替换 | 应用内自动更新 |

#### 升级建议
1. **备份配置**：导出旧版本配置（如果有重要设置）
2. **卸载旧版**：删除旧版便携版文件夹
3. **安装新版**：下载 MSI 安装包双击安装
4. **配置迁移**：如有遗留配置，使用导入功能迁移

#### 开发者变更
- 新增开发者模式启动参数
- 新增开发者诊断页面入口
- TRACE 日志支持详细诊断输出


💡 **升级提示**：v2.0.0 起统一使用 MSI 安装包，旧版便携版用户请手动迁移配置后重新安装！
